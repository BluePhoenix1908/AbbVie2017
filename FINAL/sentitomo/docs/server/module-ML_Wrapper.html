    (inner) detectSentimentEnsembleR(message) → {Promise.<String>}

    Detects the sentiment of a message using six different sentiment analysis algorithms and pick the majority result of those. It uses:

    • Classification and Regression Trees (CART)
    • Random Forest (RF)
    • Support Vector Machines (SVM)
    • Naive Bayes (NB)
    • Distributed Random Forest (DRF)
    • Gradient Boosting Machine (GBM)
