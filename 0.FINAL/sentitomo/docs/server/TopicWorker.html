(static) detectTopics() → {void}

Crawls 100 tweets from the database where the topic is not yet detected. It will then use staticBatch.py file to detect the topics with LDA. If no tweets are found, where the topics are missing, the function will wait for 10 minutes to let new tweets gets crawled and then starts again.
