(static) detectSentiments() → {void}

Crawls 100 tweets from the database where the sentiment is not yet detected. It will then use different sentiment detection algorithms to detect the sentiments If no tweets are found, where the sentiment is missing, the function will wait for 10 minutes to let new tweets gets crawled and then starts again.
